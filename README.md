# Ticky-Tacky

A modern, professional, and responsive web-based implementation of the classic Tic-Tac-Toe game. Designed for a sleek user experience with smooth gameplay and a contemporary aesthetic.

---

## 🚀 Features

- **Responsive Gameplay**: Optimized for mobile, tablet, and desktop screens.
- **Modern UI**: Styled with Bootstrap for a clean and professional look.
- **Fast Performance**: Powered by Vite and React for near-instant load times and smooth interactions.
- **Local Multiplayer**: Challenge your friends to a classic game of strategy.

## 🛠️ Technology Stack

- **Frontend**: [React 19](https://react.dev/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Styling**: [Bootstrap 5](https://getbootstrap.com/)
- **Routing**: [React Router 7](https://reactrouter.com/)

## ⚙️ Getting Started

Follow these steps to get the project running locally:

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or higher recommended)
- [npm](https://www.npmjs.com/)

### Installation & Setup

1. **Navigate to the platform directory**:
   ```bash
   cd platform
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Start the development server**:
   ```bash
   npm run dev
   ```

The application will be available at `http://localhost:5173`.
